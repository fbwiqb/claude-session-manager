📋 세션매니저 사용법

클로드/코덱스 세션을 한곳에서 찾고, 열고, 관리합니다.

🔍 찾기

  • 검색창: 이름·첫 질문으로 빠른 검색
  • 🔎본문: 켜면 대화 내용까지 검색(느릴 수 있음)
  • 소스 토글: Claude / Codex / 전체 전환
  • 정렬: 최근순·이름순·대화수(적은/많은)순

📂 열기

  • ▶️ cmd에서 열기: 원본 세션을 그대로 이어서 (claude -r). 이름·폴더 유지 — 권장
  • 🖥️ 앱으로 가져오기(복사): 데스크톱앱에 새 사본으로 import(이름 미유지)
  • 세션은 원래 작업 폴더에서 열립니다. Windows Terminal이 있으면 새 탭으로.

⭐ 관리

  • ⭐즐겨찾기: 항상 상단 고정. ⬆️⬇️로 순서 조정
  • ✏️리네임: 이름 변경(claude 피커와 동기화). 실행 중 세션은 불가
  • 체크박스 + 선택삭제: 여러 개 골라 휴지통으로
  • 🧹일괄삭제: 30일 이상 + 이름없는 세션 일괄 정리
  • 🗑️휴지통: 삭제는 휴지통 이동(복원 가능)
  • ●실행중 표시 = 지금 켜져있는 세션(보호됨)

🔄 기타

  • 🔄 새로고침 / 새 버전은 시작 시 자동 알림(업데이트)
  • 창을 닫으면 종료됩니다.
